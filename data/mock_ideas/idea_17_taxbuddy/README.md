# TaxBuddy

Tax filing software for US freelancers and 1099 contractors. Connect
your bank account, credit cards, and PayPal; TaxBuddy auto-categorizes
expenses, computes Schedule C deductions, and files your federal +
state returns directly with the IRS via authorized e-file partners.

The differentiator versus TurboTax / H&R Block: TaxBuddy is built
specifically for the 1099 lifecycle (quarterly estimated taxes, mileage
tracking, home-office deductions, retirement-account contributions). The
UI is a chat-style flow that asks one question at a time instead of a
20-page form.

Pricing: $89/year flat, no upcharges. We're profitable from year one if
we hit 5,000 paid users.

Stack: Next.js, Plaid for bank connections, GPT-4 for receipt OCR and
expense categorization, Stripe for billing, Drake Software API for IRS
e-file submission.
