# zkAttest

A zkML registry for verifiable AI model inference attestations on Solana.
A model owner registers their model weights' hash on-chain. When the
model is run on an input, a zk proof is generated that attests:
"output Y was produced by model with weights-hash H on input X." Anyone
can verify the proof on-chain in O(1) without re-running the model.

The use cases:
- AI-generated content provenance (this image was generated by *this*
  Stable Diffusion checkpoint, not a deepfake variant).
- Compliance for regulated AI (the credit decision was made by the model
  the regulator approved, not a substitute).
- Marketplaces for fine-tuned models (buyers can verify the model
  matches the demo without the seller revealing the weights).

We're using EZKL for circuit generation; proofs are ~5MB and verify in
~50ms on-chain via a Solana program. Currently demoed against a small
ResNet-18.

Stack: EZKL, Halo2, Anchor program for proof verification, PyTorch
inference harness, Next.js frontend.
